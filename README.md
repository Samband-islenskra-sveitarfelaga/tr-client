# tr-client
PowerApp Solution fyrir eyðublöð sem tengjast þjónustugátt TR. 
PowerAppið er þróað af Sambandi íslenskra sveitarfélaga fyrir þau sveitarfélög sem vilja nota það til að senda inn skjöl til þjónustugáttar TR

## Uppsetning á repo
### EndurhaefingaraetlunTR
Í möppunni EndurhaefingaraetlunTR eru skrár sem þarf til að setja upp PowerAppið í umhverfi þess sem vilja nota það. Allar breytingar og uppærslur eru checkaðar inn per skrá þannig að hver og einn getur lesið inn og unnið með einstakar breytingar. 

### docs
Leiðbeiningar og annað fyrir PowerAppið. Aðallega fyrir þá sem þróa það og setja upp.

### versions
Mappa með zip-skrám sem eru export úr þróunarumhverfi. Inniheldur þær útgáfur sem gefnar eru út af Sambandinu a.m.k.

### data
Mappa með gögnum til að lesa inn í stoðtöflur kerfis.
Listar yfir valmöguleika og annað þess háttar.
